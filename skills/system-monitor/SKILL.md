# 系统监控技能 (System Monitor)

## 技能描述
实时监控系统状态，包括 CPU、内存、磁盘、进程、网络等信息。

## 权限要求
- Level 1 及以上

## 可用命令

### 1. CPU 使用率
**命令**: `top`

**说明**: 显示当前 CPU 使用率和进程列表。

**输出信息**:
- CPU 总使用率
- 各核心使用率
- 进程列表（按 CPU 排序）
- 系统负载

**示例**:
```
top
```

---

### 2. 内存使用
**命令**: `free`

**说明**: 显示内存使用情况。

**输出信息**:
- 物理内存总量/已用/可用
- 虚拟内存（页面文件）使用
- 缓存使用情况
- 内存使用率百分比

**示例**:
```
free
```

---

### 3. 磁盘使用
**命令**: `df`

**说明**: 显示磁盘分区使用情况。

**输出信息**:
- 各分区总容量
- 已用空间
- 可用空间
- 使用率百分比
- 挂载点

**选项**:
- `-h`: 人类可读格式（自动转换单位）

**示例**:
```
df
df -h
```

---

### 4. 进程列表
**命令**: `ps`

**说明**: 显示当前运行的进程列表。

**选项**:
- `-a`: 显示所有用户的进程
- `-u`: 显示详细信息
- `-e`: 显示环境变量

**输出字段**:
- PID: 进程ID
- CPU%: CPU 使用率
- MEM%: 内存使用率
- 进程名: 可执行文件名称

**示例**:
```
ps
ps -au
```

---

### 5. 网络监控
**命令**: `netstat`

**说明**: 显示网络连接和统计信息。

**选项**:
- `-a`: 显示所有连接
- `-n`: 数字格式显示
- `-p`: 显示进程信息
- `-s`: 显示统计信息

**示例**:
```
netstat
netstat -an
netstat -s
```

---

### 6. 系统信息
**命令**: `sysinfo`

**说明**: 显示完整的系统信息。

**输出内容**:
- 操作系统版本
- 处理器信息
- 内存总量
- 计算机名
- 用户名
- 系统运行时间
- 显卡信息

**示例**:
```
sysinfo
```

---

### 7. 进程详情
**命令**: `pid <进程ID>`

**说明**: 查看指定进程的详细信息。

**参数**:
- `进程ID`: 要查看的进程 PID

**输出信息**:
- 进程名称
- 启动时间
- CPU 时间
- 内存使用
- 父进程 ID
- 线程数

**示例**:
```
pid 1234
```

---

### 8. 结束进程
**命令**: `kill <进程ID>`

**说明**: 结束指定的进程。

**参数**:
- `进程ID`: 要结束的进程 PID

**安全等级**: 5级

**选项**:
- `-9`: 强制结束

**示例**:
```
kill 1234
kill -9 5678
```

所有监控命令支持以下可视化显示：

```

内存: [████████████░░░░░░] 60%
磁盘: [██████████████████] 95%
```

### 柱状图
```

Core 0: ████████░░ 80%
Core 1: ████░░░░░░ 40%
Core 2: ██████░░░░ 60%
Core 3: ██░░░░░░░░ 20%
```

## 注意事项
1. 结束进程可能导致数据丢失，请谨慎使用
2. 系统监控会占用少量系统资源
3. 某些系统信息可能需要管理员权限
4. 网络监控需要相应的网络访问权限
