# 网络工具技能 (Network Tools)

## 技能描述
提供各种网络相关工具，包括 ping、HTTP 请求、下载、DNS 查询、端口扫描等。

## 权限要求
- Level 2 及以上

## 可用命令

### 1. Ping 测试
**命令**: `ping <目标地址>`

**说明**: 测试与目标主机的网络连通性。

**参数**:
- `目标地址`: IP 地址或域名

**选项**:
- `-n <次数>: 发送数据包次数（默认4次）
- `-t`: 持续 ping 直到停止
- `-l <大小>: 数据包大小

**输出信息**:
- 往返时间（最小/最大/平均）
- 丢包率
- TTL 值

**示例**:
```
ping google.com
ping 192.168.1.1
ping -n 10 baidu.com
```

---

### 2. HTTP 请求
**命令**: `curl <URL>`

**说明**: 发送 HTTP 请求并获取响应。

**参数**:
- `URL`: 目标 URL 地址

**选项**:
- `-X <方法>: HTTP 方法（GET/POST/PUT/DELETE）
- `-H <头部>: 添加请求头
- `-d <数据>: POST 数据
- `-o <文件>: 保存到文件
- `-v`: 详细输出模式

**示例**:
```
curl https://api.example.com/data
curl -X POST -d "name=test" https://api.example.com/create
curl -o page.html https://example.com
```

---

### 3. 文件下载
**命令**: `wget <URL>`

**说明**: 下载文件到本地。

**参数**:
- `URL`: 文件的 URL 地址

**选项**:
- `-O <文件名>: 指定输出文件名
- `-c`: 断点续传
- `-q`: 静默模式

**示例**:
```
wget https://example.com/file.zip
wget -O archive.zip https://example.com/download
```

---

### 4. DNS 查询
**命令**: `nslookup <域名>`

**说明**: 查询域名的 DNS 解析结果。

**参数**:
- `域名`: 要查询的域名

**选项**:
- `-type=<类型>: 查询类型（A/AAAA/MX/NS/TXT/CNAME）
- `server`: 指定 DNS 服务器

**输出信息**:
- 域名对应的 IP 地址
- DNS 服务器信息
- 解析时间

**示例**:
```
nslookup google.com
nslookup -type=MX gmail.com
nslookup example.com 8.8.8.8
```

---

### 5. 端口扫描
**命令**: `portscan <目标> <端口范围>`

**说明**: 扫描目标主机的开放端口。

**参数**:
- `目标`: IP 地址或域名
- `端口范围`: 单个端口或范围（如 1-1000）

**选项**:
- `-t <超时>: 超时时间（毫秒）
- `-p <协议>: 协议（TCP/UDP）

**安全等级**: 4级

**示例**:
```
portscan 192.168.1.1 1-1024
portscan localhost 80,443,8080
```

---

### 6. 网络接口信息
**命令**: `ifconfig`

**说明**: 显示网络接口配置信息。

**输出信息**:
- 接口名称
- IP 地址（IPv4/IPv6）
- MAC 地址
- 子网掩码
- 默认网关
- 收发数据包统计

**示例**:
```
ifconfig
```

---

### 7. 路由追踪
**命令**: `tracert <目标地址>`

**说明**: 追踪数据包到目标的路由路径。

**参数**:
- `目标地址`: IP 地址或域名

**输出信息**:
- 每跳的 IP 地址
- 每跳的延迟时间
- 跳数统计

**示例**:
```
tracert google.com
tracert 8.8.8.8
```

---

### 8. 网络速度测试
**命令**: `speedtest`

**说明**: 测试网络上传和下载速度。

**输出信息**:
- 下载速度（Mbps）
- 上传速度（Mbps）
- 延迟（ms）
- 抖动（ms）
- 测试服务器信息

**示例**:
```
speedtest
```

---

### 9. 查看 IP 地址
**命令**: `myip`

**说明**: 查看本机的公网 IP 地址。

**输出信息**:
- 公网 IPv4 地址
- 公网 IPv6 地址（如有）
- 地理位置（国家/城市）
- ISP 信息

**示例**:
```
myip
```

## 注意事项
1. 端口扫描可能被目标系统视为攻击行为，请仅在授权范围内使用
2. 下载大文件可能需要较长时间
3. 某些网络操作可能需要管理员权限
4. 网络速度测试会消耗流量
5. 请遵守相关法律法规，不得用于非法用途
