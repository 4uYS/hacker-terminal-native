# 文件管理技能 (File Manager)

## 技能描述
提供完整的文件和目录管理功能，包括复制、移动、删除、重命名、创建目录等操作。

## 权限要求
- Level 3 及以上

## 可用命令

### 1. 复制文件/目录
**命令**: `cp <源路径> <目标路径>`

**说明**: 复制文件或目录到目标位置。

**参数**:
- `源路径`: 要复制的文件或目录
- `目标路径`: 目标位置

**示例**:
```
cp file.txt backup.txt
cp src/ dist/
```

---

### 2. 移动文件/目录
**命令**: `mv <源路径> <目标路径>`

**说明**: 移动文件或目录到目标位置，也可用于重命名。

**参数**:
- `源路径`: 要移动的文件或目录
- `目标路径`: 目标位置或新名称

**示例**:
```
mv old.txt new.txt
mv files/ archive/
```

---

### 3. 删除文件/目录
**命令**: `rm <路径>`

**说明**: 删除文件或目录。危险操作，需要确认。

**参数**:
- `路径`: 要删除的文件或目录

**安全等级**: 3级

**示例**:
```
rm temp.txt
rm old_files/
```

---

### 4. 创建目录
**命令**: `mkdir <目录名>`

**说明**: 创建新的目录。

**参数**:
- `目录名`: 要创建的目录名称

**示例**:
```
mkdir projects
mkdir docs/reports
```

---

### 5. 重命名
**命令**: `rename <旧名称> <新名称>`

**说明**: 重命名文件或目录。

**参数**:
- `旧名称`: 原文件/目录名
- `新名称`: 新文件/目录名

**示例**:
```
rename draft.txt final.txt
rename old_project new_project
```

---

### 6. 查看文件大小
**命令**: `size <路径>`

**说明**: 查看文件或目录的大小。

**参数**:
- `路径`: 文件或目录路径

**示例**:
```
size document.pdf
size downloads/
```

---

### 7. 批量重命名
**命令**: `batch-rename <目录> <模式> <替换>`

**说明**: 批量重命名目录中的文件。

**参数**:
- `目录`: 目标目录
- `模式`: 要匹配的文件名模式
- `替换`: 替换后的模式

**示例**:
```
batch-rename photos/ .JPG .jpg
batch-rename docs/ report_ document_
```

## 注意事项
1. 删除操作不可撤销，请谨慎使用
2. 批量操作前建议先备份
3. 路径支持相对路径和绝对路径
4. 文件名包含空格时请用引号包裹
